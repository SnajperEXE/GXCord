# GXCord

> A modern, highly customizable Discord theme inspired by Opera GX.

GXCord transforms Discord with a sleek gaming-inspired interface, bold accent colors, geometric UI elements, and extensive customization. Designed around CSS variables, it lets you personalize the entire theme in minutes without digging through thousands of lines of code.

---

## ✨ Features

- 🎨 Easy color customization via CSS variables
- 🖼️ Custom wallpaper support
- 🔺 Geometric UI inspired by classic Opera GX
- 💎 Custom mention badges
- ⬡ Unique server icon shapes
- 🌈 Gradient support where applicable
- 📦 Clean and organized code
- ⚡ Lightweight and easy to customize

---

## 💡 Inspiration

GXCord is inspired by the classic visual identity of Opera GX while introducing its own design language through:

- Angular UI
- Chamfered components
- Bold accent colors
- Modern gaming aesthetic
- Extensive customization

It is **not** intended to be a pixel-perfect recreation.

---

## 🚀 Installation

1. Install Vencord or BetterDiscord.
2. Download `theme.css`.
3. Place it inside your Themes folder.
4. Enable **GXCord**.
5. Customize the variables to your liking.

---

## ❤️ Why GXCord?

Most themes only change colors.

GXCord lets you build your own theme by exposing the important parts through simple CSS variables, making customization fast, clean, and accessible.
